# QDII-fund-scout

QDII 基金限购查询工具。一键查看你持有的 QDII 基金的**申购限额、收益率、费率、回撤**，支持推送到飞书或企业微信群。

> **数据都在本地，不会上传到任何服务器。** 查询直连天天基金和好买基金的公开接口，你的基金配置和 Webhook 地址只保存在你自己的电脑上。

---

## 特性

| 特性 | 说明 |
|------|------|
| **多源交叉验证** | 天天基金主页面 + NAV API 独立计算 + 好买基金对比，三路确认数据准确；净值超过 7 天未更新时自动提示 "限额可能已变化" |
| **主备自动切换** | 天天基金不可用时自动降级到好买基金，两者都失败返回"数据暂不可用"，不会崩溃 |
| **多渠道推送** | 查询结果一键推送到飞书或企业微信群，手机端也能看 |
| **每日定时推送** | 设置后电脑自动按计划查询并推送到群里，起床/收盘即可查看 |
| **可视化界面** | 浏览器图形化操作（http://localhost:8765），添加基金、配置推送、点击查询，无需记忆命令 |
| **命令行 CLI** | 支持单只详情、批量对比、JSON / CSV / Markdown 导出 |
| **Python API** | 可集成到自己的脚本或自动化流程中 |

## 快速开始

### 安装

**方式1：一行命令安装（推荐，无需 git）**

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/hangliuc/QDII-fund-scout-skill/main/install.sh)
```

复制上面这行命令，粘贴到终端（Terminal）按回车即可。安装完成后输入 `fund-scout` 即可运行。

**方式2：git clone 安装**

```bash
git clone git@github.com:hangliuc/QDII-fund-scout-skill.git
cd QDII-fund-scout-skill
bash setup.sh
```

安装过程：自动检测 Python 环境 → 安装依赖 → 可选配置推送 → 验证安装。

### 运行

```bash
bash run.sh
```

菜单界面：

```
  1) 打开可视化界面（浏览器配置）
  2) 查看我的基金（使用配置文件）
  3) 编辑我的基金列表
  4) 手动输入基金代码查询
  5) 查询我的基金并推送到飞书
  6) 查询我的基金并推送到企业微信
  7) 查询我的基金并同时推送飞书 + 企业微信
  8) 配置推送渠道（飞书/企业微信）
  9) 设置每日定时推送（自动运行）
  10) 取消定时推送
  0) 退出
```

> **首次使用**：先选 **3) 编辑我的基金列表**，输入基金代码和名称（6 位数字，名称可自定义）。然后选 **2) 查看我的基金** 或 **1) 打开可视化界面**。

## 配置推送

**非必需**，不配置也能正常使用。配置后可在手机端接收数据。

### 飞书配置

1. 打开飞书 App，进入任一你拥有管理权限的群聊
2. 点击群聊右上角 **...** → 选择 **群机器人** → **添加机器人**
3. 搜索并选择 **Webhook 机器人**，点击 **添加**
4. 复制机器人的 Webhook 地址（以 `https://open.feishu.cn/open-apis/bot/v2/hook/` 开头）
5. 运行 `bash run.sh` → 选 **8) 配置推送渠道** → 粘贴地址并保存
6. 可在菜单中选择 **5)** 测试推送

### 企业微信配置

1. 打开企业微信 App，进入任一群聊
2. 点击群聊右上角 **...** → 选择 **群机器人** → **添加机器人**
3. 点击 **新创建一个机器人**，输入机器人名称（如 "QDII 数据推送"）
4. 复制 Webhook 地址（以 `https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=` 开头）
5. 运行 `bash run.sh` → 选 **8) 配置推送渠道** → 粘贴地址并保存
6. 可在菜单中选择 **6)** 测试推送

## 每日定时推送

配置好基金列表和推送渠道后，可以设置电脑在固定时间自动查询并推送到群里：

```bash
bash run.sh
# 选 9) 设置每日定时推送
# 可选：工作日 09:00 / 15:30 / 每天 09:00
```

> - 电脑在计划时间需保持开机并联网
> - Mac 合盖睡眠时定时任务不会触发
> - 日志路径：`~/.fund-scout/schedule.log`

## Agent / Skill 使用

本项目可作为 AI Agent（如 SOLO、Trae）的 Skill 使用。向 Agent 提问示例：

```
帮我用 QDII-fund-scout 查一下 008971 的申购限额和近一年收益

获取我的基金列表数据，生成对比表格，推送到飞书群

每天早 9 点自动查 QDII 基金限额，有变化时重点标注
```

Agent 会自动读取 `SKILL.md` 中的规则和引用文件，执行数据获取、格式化、推送等操作。

## CLI 高级用法

```bash
cd scripts

# 查看单只基金详情
python3 cli.py detail 012870

# 批量对比多只基金
python3 cli.py compare 012870,006479,008971

# 使用配置文件中的基金列表
python3 cli.py compare --config ~/.fund-scout/config.json

# 推送到飞书 / 微信
python3 cli.py compare 012870,006479 --push feishu
python3 cli.py compare 012870,006479 --push wechat

# 导出格式
python3 cli.py compare 012870,006479 --format json
python3 cli.py compare 012870,006479 --format csv
```

### Python API

```python
from core.fetcher import FundFetcher

fetcher = FundFetcher()
result = fetcher.compare(["012870", "006479"])

for fund in result.funds:
    print(f"{fund.name}: {fund._purchase_info}  近1年={fund.return_1y}")
```
